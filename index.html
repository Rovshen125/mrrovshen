Saytların Hazırlanması

Biz "mrrovshen design & study" olaraq sizin istəklərinizi tam yerinə yetirməklə, istədiyiniz dizaynda olan veb saytarı sürətli şəkildə hazırlayıb və təhvil veririk. Hazırladığımız saytlar responsive dizayna və sürətli hostinqlərlə təmin olunur.

Tədris və Təlim

Proqramlaşdırma Dərsləri

4-11 ci sinif şagirdləri üçün front-end və oyun proqramlaşdırma bizimlə daha marağlı olacaq. Tələbələr qısa zaman ərzində öz veb saytların hazırlayacaq və pulsuz olaraq domen və hostinqlə təmin ediləcəklər. Eyni zamanda Javascript vəastəsi ilə öz oyunlarınıda sərbəst şəkildə kodlaya biləcəklər.

HTML

Tələbələr ilk öncə HTML 5 ilə tanış olub kodlamadının məntiqi ilə tanış olacaqlar. Bu onlarda daha asan kod yazma vərdişləri yaradacaq.

CSS

CSS vəastəsi artıq yeni bir dil ilə tanış olacaqlar. Bu onlara hazırladığları proqramları daha da gözəl hazırlamağa imkan verəcək.

Javascript

Javascript ilə artıq tələbələr yüksək səviyyəli tanış olacaqlar. Bu onlara sürətli şəkildə oyun hazırlamağada köməklik edəcək.

Veb Dizayn

Veb Dizayn Kursları
